创建github仓库

下载工具PicGo

GitHub PicGo

PicGo是一个用于快速上传图片并获取图片URL链接的工具

获取仓库 access tokens

Settings -> Developer settings -> Personal access tokens

点击 Generate new token 获取token

将token保存至安全的地方,这里的token只会显示一次（务必保存好）

配置PicGo中的GitHub图床

查看图床列表

使用其他图床

也可以使用其他图床，比如sm.ms
sm.ms 有5GB的免费空间，短时间内是够用的，速度也还可以

  • 注册后在控制台获取token
  • 在PicGo中配置即可
  • 上传时在这里选择源就行
